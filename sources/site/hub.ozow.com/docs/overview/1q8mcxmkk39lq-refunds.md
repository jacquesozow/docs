# Refunds

**At Ozow we want to take the hassle out of the refund process.**

Ozow Refunds allows customers to receive refunds from merchants instantly with a few simple clicks. Using the same platform that the purchase was made on, Ozow Refunds provides a seamless payment process without the need to request any customer or banking information.

Read more about Ozow Refunds [here](https://ozow.com/our-products/refunds).

## [How to process a Refund](https://hub.ozow.com/#how-to-process-a-refund)

1. User logs onto the Merchant Dashboard
2. User navigates and selects the Transactions menu and searches for the transaction which needs to be refunded with Status = Complete
3. User selects the transaction from the transaction list
4. User selects the Refund or Manual Refund button
5. User completes the Refund details and clicks Submit button
6. User navigates and selects the Refund menu
7. User views the Refunds list where all the Refunds will be displayed

> **Split refunds**
> 
> Partial refunds can be processed where only a part of the total transaction amount gets refunded. It is possible to process multiple refunds on a single payment, up to the total transaction amount.
