## Self-service via the Merchant Dashboard

The [Merchant Dashboard](https://dash.ozow.com) is your central hub for account management. Most common tasks don't require a support ticket — you can handle them yourself in minutes:

| Task | Where to find it |
|---|---|
| View transaction history | **Transactions** in the main menu |
| Process a refund | **Transactions** → select transaction → **Refund** |
| Top up your float | **Floats** → **Top Up Float** |
| Set a float balance alert | **Merchant Details** → **Edit Details** → Float Balance Alert Amount |
| Manage sites | **Sites** in the main menu |
| View or regenerate API keys | **Sites** → select site → **API Keys** |

For a full walkthrough of float management, including step-by-step top-up instructions and alert configuration, see the [Float Management](/help/float-management) page.

## Testing and staging environment

If you're building or debugging an integration, use the **Staging Dashboard** at [stagingdash.ozow.com](https://stagingdash.ozow.com) to test your flows against Ozow's sandbox environment without affecting live transactions. The staging environment mirrors the production dashboard and supports both Mock API and Actual API testing for payouts.

## Plugin and integration partner queries

<Note>
  If your query relates to a specific e-commerce plugin (such as WooCommerce, Shopify, or Magento) or an integration partner platform, please email [support@ozow.com](mailto:support@ozow.com) and include your **platform name**, **plugin version**, and a brief description of the issue. This helps the team route your query to the right specialist quickly.
</Note>
